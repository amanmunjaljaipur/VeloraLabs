# Industry leaders → App Builder questions

What we learned from leaders and how Verlin App Builder maps it.

## Shopify
| They ask / do | We ask / do |
|---------------|-------------|
| Online vs offline / where you sell | `sellChannel` |
| What you sell | `whatYouSell` + product CMS |
| Shipping | `shippingHow` → deliveryNote + FAQ |
| Payments | `paymentToday` → paymentMethods |
| Goals / setup checklist | `successGoal` + launch checklist |
| Product images | Find photos + auto custom images |

## Wix AI / ADI
| They ask / do | We ask / do |
|---------------|-------------|
| Business type & name | idea cards + `brandName` |
| What makes you unique | `uniqueSelling` |
| Website goal | `appHelpHope` |
| Design / logo | `logoPreference` + city colours |

## Dukaan / Instamojo (India MSME)
| They ask / do | We ask / do |
|---------------|-------------|
| WhatsApp catalog | WhatsApp CTAs, order methods |
| UPI / COD | `paymentToday` chips |
| Shareable store link | public `/apps/{slug}` + shareWhere |
| Simple mobile-first store | standalone shell, mobile nav |

## Square / local retail
| They ask / do | We ask / do |
|---------------|-------------|
| Staff | `whoDoesWhat` + Team roles |
| Peak hours | `busyTimes` + openingHours |
| In-person + online | offline workflow + sellChannel |

## Product questions that raise quality
1. Offline day & customer steps (jobs-to-be-done)
2. Channels (where they already sell)
3. USP (why buy from them)
4. Concrete product names (for search + custom images)
5. Fulfilment + payment (trust + FAQ)
6. 2–3 website jobs + 3-month success
7. Where they will share the link
8. Logo own vs design

Never ask enterprise jargon. Always chips + free text.
